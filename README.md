## 📋 Sobre o Projeto

Desafio proposto pela **DIO (Digital Innovation One)** na formação de lógica de programação. O objetivo é criar um classificador que determina o nível de um herói com base na quantidade de experiência (XP) acumulada.

Este projeto demonstra o uso de conceitos fundamentais de programação:
- ✅ Variáveis
- ✅ Operadores
- ✅ Estruturas de decisão (if/else)
- ✅ Laços de repetição
- ✅ Entrada e saída de dados

---

## 🎯 Objetivo

Desenvolver um sistema que classifique heróis em diferentes níveis baseado em sua experiência (XP), seguindo a tabela de classificação:

| XP                  | Nível       |
|---------------------|-------------|
| Menor que 1.000     | Ferro       |
| 1.001 - 2.000       | Bronze      |
| 2.001 - 5.000       | Prata       |
| 5.001 - 7.000       | Ouro        |
| 7.001 - 8.000       | Platina     |
| 8.001 - 9.000       | Ascendente  |
| 9.001 - 10.000      | Imortal     |
| 10.001 ou mais      | Radiante    |

---

## 🚀 Funcionalidades

- 🎭 Entrada do nome do herói
- 📊 Entrada da quantidade de XP
- 🏆 Classificação automática do nível
- 💬 Exibição de mensagem personalizada
- 🔄 Possibilidade de classificar múltiplos heróis
